What I learned in Fundamentals of Web Design

I must say even though my motivation did burn out about 1/4 of the time into this term, that I still did learn a lot about HTML and CSS. At the very beginning of the class, we were learning how to code on our own using a website called freecodecamp.com and it taught us how to use style tags, headers, footers, change the background, and so on. I believe this was a good starter to the class to let us explore on our own instead of using Lan School and just showing us certain tags that do certain things, like paragraphs or url links.

A brief summary of things I learned

  • line break
  • insert images / videos
  • font style / color
  • borders
  • tables
  • headers / footers
  • titles
  • open new windows via link
  • url links attached to words or images
  • inserting youtube videos
  • paragraphs
